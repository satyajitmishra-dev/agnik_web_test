## Features
- **Circle Animation**: Animate circles in different colors (Red, Green, Blue)
- **Customizable Timing**: Set animation duration for each color
- **Adjustable Circle Count**: Control the number of circles (1-10)
- **Responsive Design**: Works on desktop and mobile devices

## How to Use
1. Set the duration (in seconds) for Red, Green, and Blue circle animations
2. Enter the number of circles you want to generate (1-10)
3. Click **Submit** to create the circles
4. Click **Start** to begin the animation
